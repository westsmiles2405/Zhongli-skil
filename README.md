# 🪨 钟离工作陪伴 (Zhongli Companion)

> 让岩王帝君钟离以沉稳契约精神陪伴你的 VS Code 工作之旅。

## 功能

| 命令 | 说明 |
|------|------|
| `钟离：今日契约` | 开场问候，开启今日工作 |
| `钟离：开始番茄钟专注` | 启动番茄钟（默认 25 分钟） |
| `钟离：结束番茄钟` | 手动停止番茄钟 |
| `钟离：给我打气` | 获得钟离的沉稳鼓励 |
| `钟离：契约审判代码` | 让钟离审视你的代码质量 |
| `钟离：今日收工` | 结束今日工作，回顾番茄钟完成数 |
| `钟离：讲个璃月典故` | 听钟离讲璃月的历史与故事 |
| `钟离：沏一壶好茶` | 让钟离泡一壶茶，暂时放松 |
| `钟离：聊聊胡桃` | 聊聊往生堂堂主胡桃 |

## 特色

- **契约精神**：以岩王帝君的契约哲学激励工作
- **品茶论道**：工作间隙沏茶放松，感受璃月的宁静
- **璃月典故**：聆听六千年的历史故事
- **闲置提醒**：长时间未操作时温和提醒
- **打字机效果**：沉稳的逐字输出，契合钟离的从容不迫

## 设置

| 选项 | 默认值 | 说明 |
|------|--------|------|
| `zhongli.pomodoroMinutes` | 25 | 番茄钟专注时长 |
| `zhongli.breakMinutes` | 5 | 休息时长 |
| `zhongli.enableIdleReminder` | true | 闲置提醒开关 |
| `zhongli.idleMinutes` | 30 | 闲置多少分钟后提醒 |
| `zhongli.enableStatusBar` | true | 状态栏显示 |

## 安装

```bash
# 从 .vsix 安装
code --install-extension zhongli-companion-0.1.0.vsix
```
